## What Drives Mini-Split Installation Cost Higher

### 1. Line Set Length

A standard installation runs the refrigerant line set up to 25 feet from outdoor unit to indoor air handler. Each additional 10 feet adds $150–$300.

- Standard run (under 25 ft): included in base price
- Medium run (25–50 ft): +$150–$400
- Long run (50–100 ft): +$400–$900

### 2. Electrical Work

Mini-splits require a dedicated 240V circuit. If one doesn't exist (common in older homes and additions), add:
- New 240V circuit from panel: $200–$500
- Panel upgrade (if needed): $1,500–$3,500

### 3. Installation Complexity

- **Standard wall installation**: Included in base price
- **Ceiling cassette installation**: +$300–$700 (recessed, less visible)
- **Ducted mini-split** (concealed air handler): +$500–$1,200
- **Second-story or attic installation**: +$200–$500 (scaffolding, difficult access)

### 4. Multi-Zone System Design

Multi-zone systems use a single outdoor condenser connected to multiple indoor units. The condenser must be sized for the total connected indoor capacity. Mismatched systems are a common installer error — insist on proper system design documentation.

## Mini-Split vs. Central HVAC: When Each Makes Sense

**Choose a mini-split when:**
- No ductwork exists and adding ducts would cost $5,000–$15,000
- You're conditioning a room addition, garage, sunroom, or ADU
- You want independent temperature control by room
- You're replacing window AC units and baseboard heat in an older home
- You live in a mild-to-moderate climate and want highly efficient operation

**Choose central HVAC when:**
- You already have good ductwork throughout the home
- You're replacing both heating and cooling in a ducted home
- You prefer single-system maintenance
- Budget favors the lower cost of matching ductwork to existing infrastructure

## Rebates for Mini-Split Systems in 2026

Most utility heat pump rebates apply to mini-split systems as well as central heat pumps:

- **Utility rebates**: $100–$1,000 per zone in many service areas
- **DOE HEAR program**: Up to $8,000 for income-qualified households
- **State programs**: Varies by state — same programs as central heat pumps

See our [heat pump rebates by state guide](/blog/heat-pump-rebates-by-state) for state-by-state details. Mini-splits and ducted heat pumps are generally treated the same for rebate purposes.
